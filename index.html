Git commands (Mac)

  • git init -> create local git repo
  • git init . -> specifies the current folder on init
  • ls -al -> list all files including hidden files/folders
  • git status -> shows modified files
  • git add {filename} -> add file to staging area
  • git commit -m "add commit message here" -> commit file with message
  • cd .git/ ->(not recommended to go into .git folder)
  • rm -rf .git -> remove .git folder and repo
  • git add . -> add all modified files to staging area
  • git log -> logs all commits
  • git show -> shows the last commit with all its changes
  • git ls-files -> shows all files being trackes by git
  • git commit -a -> adds modified files to the staging area and also commits the files
  • git commit -am "add commit message here" -> sames git commit -a but adds message
  • git reset HEAD {filename} -> unstage file
  • git checkout -- {filename} -> revert changes to last known commit
  • git help log -> extensive overview of git commands
  • git log --oneline --graph --decorate --all -> compact history overview of commits
  • git config --global alias.hist "log --oneline --graph --decorate --all" -> create hist alias for overview command
  • git config --global --list -> overview of git alias commands
  • git hist -> compact history overview of commits with alias
  • git hist -- {filename} -> compact history overview of commits of file with alias
  • git mv {filename} {filename} -> rename file
  • gir rm {filename} -> remove file
  • git add -u -> git sees a file rename as a deletion and a rename, for a deletion update git with this command
  • git add -A -> to update git about both additions and deletions
  • .gitignore -> exclude file from being tracked by git. (this folder must be tracked in the git repo)
  • git diff {git hist commit id} HEAD -> compare difference of commit to latest commit
  • git diff -> compare difference of working directory to latest commit
  • git help diff -> overview of git diff commands
  • git branch -> shows branches and highlites current branch
  • git checkout -b {branch name} -> creates new branch and switches to this branch (adopts current pending changes)
